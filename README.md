# Data Science Capstone — IBM / Coursera

Coursework completed as part of the IBM Data Science Professional Certificate on Coursera. The capstone project applies data collection, querying, exploratory analysis, and visualization techniques to a real-world dataset tracking SpaceX Falcon 9 launch and landing outcomes.

## Tools & skills covered

- **Python** (Pandas, Numpy) — data manipulation and analysis
- **SQL** (SQLite, Db2) — querying and aggregating structured data
- **Web scraping** (BeautifulSoup, Requests) — collecting data from live sources
- **Exploratory data analysis (EDA)** — identifying patterns and distributions in data
- **Geospatial visualization** (Folium) — mapping launch sites and outcomes
